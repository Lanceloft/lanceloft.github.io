开头介绍

  我的第一次搭建博客大约在五个月前，同样利用hexo，当初对于npm,node等都不熟悉，所以勉强用了挺久的时间才搭出第一次的hexo博客。甚至说再建一次也要重新跟着教程跑才能写出教程过了几个月知识广了，现在重新搭建大约就十来分钟，甚至包括npm intsall的时间。所以就从头开始吧。

  首先不建议在windows下进行，不仅因为Windows的cmd太丑了，也因为windows下使用node会有各种不友好。所以建议使用mac os，当然买不起mac的就用linux系统，现在的ubuntu十分人性化，十分适合不需要设计的编程or就上上网的人。

正式开始

  首先hexo是基于nodejs的，其他的介绍可以在官网看到就不bb了。首先需要安装node和npm。可以在官网进行下载。建议使用nvm。

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.32.1/install.sh | bash

如果终端没报错的话，可以继续执行nvm install stable

接下来使用nvm use node，如果终端提示找不到node，则重启终端。再次运行。

这时候运行

node -v

以及

npm -v

如果都能看到版本，证明node和npm安装成功。接下里使用以下命令全局安装hexo。

npm install -g hexo-cli

如果终端没翻墙会发现很慢也不知道有没有卡住，使用以下命令会有点心理安慰

npm install -g hexo-cli -d

接下来进入任意一个文件夹，运行

hexo init blog

博客已初始化，当然现在还不能用运行

cd blog

npm install -d

稍等片刻依赖包就已经安装好了

hexo server

在浏览器输入localhost:4000就可以看到自己的博客建成了。是不是瞬间成就感满满的，接下来还会继续介绍hexo的各种神奇功能。
