利用hexo建博客1

  • Lanceloft
  • 1 Minutes
  • December 2, 2016

　　昨天已经介绍了利用hexo搭建默认的博客。那么今天来介绍更改hexo的主题。hexo的主题在官网上有着非常多漂亮的demo 这是hexo主题链接。
　　https://hexo.io/themes/

　　以我现在用的laughing为例，其他主题的安装也是类似的。官网文档在这里
　　https://github.com/BoizZ/hexo-theme-laughing

首先还是在blog目录下执行

npm install hexo-renderer-pug –save

安装成功后，可以开始下载主题

cd themes
git clone https://github.com/BoizZ/hexo-theme-laughing.git

当git执行成功后主题已经成功下载到我们的hexo里面了，接下来只需要在修改blog目录下面的_config.yml文件。
例如使用ubuntu自带的gedit编辑器可以

gedit _config.yml

在 Extensions的theme可以看到默认的theme写着landscape。我们只需要修改为hexo-theme-laughing就可以。例如
theme: hexo-theme-laughing
注意冒号后的空格。yml里面的书写必须非常规范。接下来运行hexo s就可以看到已经使用了新的主题了。
其他主题的使用方法也是一样，根目录npm install 依赖 -> 进入theme clone 主题的地址 -> 编辑根目录的theme配置。
