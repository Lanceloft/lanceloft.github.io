如果还是看不懂的伸手党就继续往下看把

shadowsocks-4.0.4下载地址

  1. 安装后在右下角任务栏找到shadowsocks图标
  2. 添加服务器节点,如果没有,这里是网上分享的站点
  3. 选择 启用系统代理 启用系统代理就好

这个是最基础的使用方法,当然还有进阶使用例如pac等等可以参考文档这里不做介绍了。ps.科学上网有风险
